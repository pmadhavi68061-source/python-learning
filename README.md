# DNA Base Counter 🧬

A Python script that counts A, T, G, C nucleotide bases 
in a DNA sequence.

## How to Run
python dna_base_counter.py

## Example

Input:  ATCGGCTA

Output:
A: 2
T: 2
C: 2
G: 2
Total bases: 8

- Python functions
- Dictionaries
